== Description ==

= Get a Popup Modal Windows in a Widget for your Wordpress site! =
<p>Simple popup to add your offers, promotional content via a widget.</p> 
<p>HTML is allowed in the Content area of the popup, thus you can add images, styling, etc.</p>
<p>You define a Header if needed, Footer if needed, the Content and Modal Window Width.</p> 

**PRO VERSION**

* Choose if you will show again Popup to Users that clicked Close Button of Popup.
* Color Picker for Title Background.
* Color Picker for Title Color.
* HTML wrapper for Title (choose between h1,h2,h3,h4,h5 headings, p or strong).
* Color Picker for Content Background.
* Color Picker for Content Color.
* Color Picker for Footer Background.
* Color Picker for Footer Color.
* HTML wrapper for Footer (choose between h1,h2,h3,h4,h5 headings, p or strong).
* Display Any Content with a shortcode in Content Area of Popup.

[Get it Here](https://webdeveloping.gr/product/wordpress-modal-widget-pro "Webd Wordpress Modal Widget")

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins` directory and unzip, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go and use it in the widget screen
4. Add header, content, footer, asign it to a region and it work for you out of the box!
